# OurC-interpreter
一個約5000行的類C語言直譯器(可執行迴圈和functions)
此為個人作品。<br>
<br>
<br>

## 開發平台
Windows10<br>
<br>
<br>

## 開發環境
Dev C++<br>
<br>
<br>

## 使用方法
第一行須輸入數字(限整數)作為test number用<br>
enter後將顯示互動式介面資訊<br>
接著可輸入C語言之簡單程式碼<br>
此介面會直譯並檢查語法是否有錯<br>
接著執行該程式碼之動作，並顯示相關結果<br>
目前支援之語法包含if、else、while、cout、define function等等<br>
